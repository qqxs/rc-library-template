# rc-library-template

![workflow](https://github.com/qqxs/rc-library-template/actions/workflows/cl.yml/badge.svg)

## Use

```bash
# build production
yarn run build

# run fix:prettier fix:eslint
yarn run fix

# jest test

yarn run test

# development
yarn run storybook

# build storybook docs

yarn run storybook:build

# npm publish

yarn run publish
```

## version

node >= 10

react >= 16.8
